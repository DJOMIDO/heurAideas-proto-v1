# HeurAIDEAS

> A collaborative project management platform for heuristic evaluation research teams.

---

## Method 1: Docker One-Command Start (Recommended)

**The simplest way** - Start all services (frontend + backend + database) with one command.

### Prerequisites
- [Docker Desktop](https://www.docker.com/products/docker-desktop/)

### Start Command
```bash
git clone https://github.com/DJOMIDO/heurAideas-proto-v1.git
cd heurAideas-proto-v1
docker-compose up -d
